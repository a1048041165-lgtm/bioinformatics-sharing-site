Molecular Docking

Molecular docking predicts the preferred orientation of one molecule to a second when bound to form a stable complex. Tools like AutoDock Vina are widely used to dock small molecules into macromolecular targets. Typical workflows involve preparing the receptor and ligand, performing docking, and evaluating binding modes. For example, the AutoDock Vina documentation walks through docking the anticancer drug imatinib (Gleevec) into the kinase domain of c Abl.

Useful resources for learning and performing molecular docking:

  • Autodock 分子对接详细步骤 (CSDN blog) – Step-by-step guide for performing molecular docking with AutoDock.
  • AutoDock Vina Basic Docking Tutorial – Official documentation describing how to prepare receptors and ligands and run docking.
  • AutoDock Vina 使用教程 (CSDN blog) – Detailed tutorial on using AutoDock Vina.
  • Basic docking — Autodock Vina documentation – Example of docking imatinib into c-Abl with explanations of the workflow.
  • MGLTools Downloads – Download tools required for preparing receptors and ligands (e.g., AutoDock Tools).
  • AlphaFold Protein Structure Database – Use for obtaining protein structures to dock against.
